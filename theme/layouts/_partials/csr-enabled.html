{{/* Returns whether CSR (client-side chrome rendering) is enabled for the site, via the td.csr_enable param. Normalizes the value because it can reach us as either type: a real boolean when set in site config (csr_enable = true), or the string "true"/"false" when set through a HUGOxPARAMSxTDxCSR_ENABLE environment variable. Hugo only casts an env override to the existing value's type when that key is already present in the site's own config; a theme-provided default isn't enough, so a bare `if (.Param "td.csr_enable")` would treat the string "false" as true. See the Client-side chrome rendering guide for what this is and how to enable it: docsy.dev/content/en/docs/content/csr.md ctx: PAGE */ -}} {{ return eq (lower (printf "%v" (.Param "td.csr_enable"))) "true" -}}
