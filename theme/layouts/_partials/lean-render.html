{{/* Returns whether the partial's REGION should render on the current PAGE under the params.td.lean_render build mode. The mode is off (every region renders) unless its value is "remove" (case-insensitive), which keeps only enough instances of each repeated, auto-generated region for a link checker to reach every link once: navbar, footer configured chrome -> each home page sidebar computed left-nav -> each locale's docs landing page The string value names the mode, leaving room for a future "tag" mode. Reading a param (not os.Getenv) means no consuming site needs a security.funcs.getenv change. See the Link checking guide for what this is and how to enable it: docsy.dev/content/en/docs/best-practices/link-checking.md ctx: dict "page" PAGE "region" REGION */ -}} {{ $p := .page -}} {{ $render := true -}} {{ if eq (lower ($p.Param "td.lean_render")) "remove" -}} {{ if eq .region "sidebar" -}} {{ $root := cond (and (ne $p.Params.toc_root true) (eq $p.Site.Home.Type "docs")) $p.Site.Home $p.FirstSection -}} {{ $render = and $root (eq $p.RelPermalink $root.RelPermalink) -}} {{ else -}} {{ $render = $p.IsHome -}} {{ end -}} {{ end -}} {{ return $render -}}
